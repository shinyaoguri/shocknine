# shocknine

給食献立オープンデータを用いた献立アプリ.
ビルドが不要で，ほぼ1つのHTMLファイルで簡潔させたアプリであることが特徴です．
Vue＋OnsenUIを利用しています．

このプロジェクトは終了しました．
データは2020年3月までのものしかありませんが，追加すれば動くはずです．
このシステムは半田市LINE公式アカウントの給食情報提供機能に引き継がれました．
